# Security Policy

## Reporting

Please report security or privacy issues privately to the repository maintainers instead of opening a public issue with sensitive details.

For public content corrections, open a GitHub issue and include the affected file, link, and suggested correction.

## Scope

This repository is a guide and prompt content surface. It does not publish API keys, private source URLs, credentials, or runnable production services.

Do not commit:

- API keys or tokens
- private source URLs
- unpublished pricing or availability claims
- internal screenshots or exports that were not approved for public use
